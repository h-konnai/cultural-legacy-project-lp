# Cultural Legacy Project LP

CLP（Cultural Legacy Project）の特設LPデモです。

## 目的
このLPは、単発の体験予約ページではなく、  
CLPという事業の「存在している構造」を見せるためのデモです。

## 構造
- 体験一覧
- 体験詳細
- 予約
- 決済（デモ）
- 体験後ページ
- 商品一覧
- CLP説明
- 参画導線

## 役割
- 体験できることを見せる
- 商品につながる流れを見せる
- CLPが文化を回す構造を見せる
- 推薦や関与の導線を薄く持たせる
